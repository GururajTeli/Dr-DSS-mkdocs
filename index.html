Overview

dss_metadata_utils is a Python package designed to handle various metadata operations related to datasets, projects, notebooks, flows, recipes, scenarios, and shared datasets. This package provides a set of utilities to streamline metadata management in data science workflows.

To organize the dss_metadata_utils code into a modular structure suitable for a Python library, we can separate different functionalities into different modules (Python files). Each module will contain related functions.
